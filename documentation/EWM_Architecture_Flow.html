3. Advanced API Request Lifecycle

A detailed sequence diagram tracking a secure API request (e.g., Requesting Leave) from the client to the database and back.
                sequenceDiagram
                    autonumber
                    participant C as Client (React)
                    participant R as Router (Express)
                    participant A as Auth Middleware
                    participant CA as Cache Middleware
                    participant CTRL as Controller
                    participant DB as MongoDB
                    participant AL as Audit Logger

                    C->>R: POST /api/v1/hr/leave (JWT in Header)
                    R->>A: Extract & Verify JWT Token
                    
                    alt Invalid Token
                        A-->>C: 401 Unauthorized
                    else Valid Token
                        A->>A: Attach User to Request (req.user)
                        A->>CA: Check Redis Cache (if GET request)
                        
                        alt Cache Hit
                            CA-->>C: Return Cached JSON
                        else Cache Miss (or POST/PUT)
                            CA->>CTRL: Proceed to Controller Logic
                            CTRL->>DB: Query/Update Document (Mongoose)
                            DB-->>CTRL: Return Result
                            
                            CTRL->>AL: Log Action (e.g., 'LEAVE_REQUESTED')
                            AL->>DB: Save Audit Log Entry
                            
                            CTRL->>CA: Invalidate Relevant Cache Keys
                            CTRL-->>C: 200 OK / 201 Created (JSON Response)
                        end
                    end

4. Finance & Payroll Processing Logic

A procedural flowchart demonstrating how the system calculates payroll automatically at the end of the month using settings and time tracking.
                stateDiagram-v2
                    direction TB
                    
                    state "Start Payroll Cycle" as Start
                    state "Fetch Global Settings" as Settings
                    state "Fetch Employee Data" as FetchEmp
                    state "Aggregate Monthly Attendance" as Att
                    state "Calculate Hours & Overtime" as CalcHours
                    state "Calculate Deductions (Taxes/Leaves)" as CalcDed
                    state "Generate Payslip Object" as GenSlip
                    state "Save to Payroll DB (Status: Pending)" as SaveDB
                    state "Finance Approves & Marks Paid" as Approve
                    state "Employee Downloads PDF Payslip" as PDF
                    
                    Start --> Settings: Initialize
                    Settings --> FetchEmp: Load Tax %, Base Hours
                    FetchEmp --> Att: For Each Employee
                    Att --> CalcHours: Sum clock-in/out diffs
                    CalcHours --> CalcDed: Apply Tax & Unpaid Leaves
                    CalcDed --> GenSlip: Base + Overtime - Deductions = Net Pay
                    GenSlip --> SaveDB: Store Record
                    
                    SaveDB --> Approve: Finance Review
                    Approve --> PDF: Available in Employee Portal
